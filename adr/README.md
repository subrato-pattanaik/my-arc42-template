# Architecture Decision Records (ADR)

This directory contains detailed ADR files for **Traffic Pursuit Unit**.

The summary table of all decisions is maintained in [../sections/09-architecture-decisions.md](../sections/09-architecture-decisions.md).

## How to add a new ADR

1. Copy `ADR-001-template.md` to `ADR-NNN-short-title.md`
2. Fill in all sections
3. Add a row to the summary table in `sections/09-architecture-decisions.md`

## Status legend

| Status     | Meaning                                   |
|------------|-------------------------------------------|
| Proposed   | Under discussion, not yet decided         |
| Accepted   | Decision made and in effect               |
| Deprecated | No longer relevant                        |
| Superseded | Replaced by a newer ADR (link to it)      |
