# Deployment View – Environments

This directory contains detailed deployment diagrams for each environment of **Traffic Pursuit Unit**.

The summary index is maintained in [../sections/07-deployment-view.md](../sections/07-deployment-view.md).

## Environments

- [Development Environment](01-development.md)
- [Staging Environment](02-staging.md)
- [Production Environment](03-production.md)

## How to add a new environment

1. Copy one of the existing files to `NN-environment-name.md`
2. Fill in the infrastructure overview and tables
3. Add a row to the summary table in `sections/07-deployment-view.md`
